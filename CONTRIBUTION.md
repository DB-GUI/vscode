# PPZ.vscode 开发手册
开发 PPZ.vscode 分以下几步：
1. 安装 [vscode](https://code.visualstudio.com/) 和 [Node.js](https://nodejs.org/en/)
2. 克隆代码仓库
``` bash
git clone git@github.com:ppz-pro/ppz.vscode.git
```
3. 安装依赖
> 由于“网络原因”，安装依赖可能会比较慢  

``` bash
cd ppz.vscode
npm install
```
4. 用 vscode 打开项目文件夹，按下键盘上的 ```f5```
