因为：

  1. 减少发行包体积（减小好几倍）
  2. 降低兼容性开发导致 bug 的几率
的原因，开发者最终决定把“PPZ.vscode 的发布”与“sqlite3 驱动”分开

第一步. 安装 Node.js

如果你的电脑上已经安装了 Node.js 则可以跳过这一步

如果没有，可以在官网下载安装包，双击即可安装

第二步. 安装 sqlite3 驱动

执行：

cd 正在检测路径...
npm install sqlite3 

注意：

  • 由于“网络原因”，sqlite3 的驱动很难安装成功，但连手机热点可以很轻松安装成功
  • 驱动安装路径在每台电脑上都不一样

完成，就这两步
