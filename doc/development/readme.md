# PPz.vscode 开发
+ [主模块介绍](./module.md)

##### 环境
+ 一台电脑（windows 或 mac 或 linux）
+ git
+ Nodejs18
+ vscode

##### 拉代码
``` bash
git clone git@github.com:ppz-pro/PPz.vscode.git
code ./PPz.vscode # 用 vscode 打开
```

##### 启动开发模式
``` bash
# 在 PPz.vscode 根目录
npm install # 安装依赖
npm run build # 打包代码
```

打包好后（应该能在 1 秒内打包好），按键盘 f5 启动开发版
